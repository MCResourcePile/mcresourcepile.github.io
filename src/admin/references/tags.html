Tags are a way of defining key words for searchable objects, such as maps, so that the user can easily find the thing they are looking for. Tags may include such things as the game mode, seasonal remix, or anything else which defines the map.

Short Tags

While tags can effectively be anything, below are a list of predefined shortened tags which increases their searchability by including keywords ontop of the tag display name. Tags not in this list will simple be capitalized when built into the page and wont include any additional keywords. Tags such as blitz or christmas will simply be converted to Blitz and Christmas on the webpage.

Display Reference Notes Type
Attack/Defend ad Gameplay
Capture Points cp Gamemode
Capture the Flag ctf Gamemode
Capture the Wool ctw Gamemode
Destroy the Core dtc Gamemode
Destroy the Core and Monument dtcm, dtmc Should be used in favor of seperate Core and Monument tags Gamemode
Destroy the Monument dtm Gamemode

Ghost Squadron gs Gamemode
King of the Flag kotf Shouldn't be mixed with ctf unless map has characteristics of both games Gamemode
King of the Hill king, koth Used when hills award points to owner Gamemode
Race for Wool rfw Gamemode
Survival Games sg Gamemode
Team Deathmatch tdm Gamemode
TNT tnt Gameplay
Tournament te, tm, tournament Gameplay
Example
"tags": ["cp", "tnt", "christmas"]
// Would become: Capture Points, TNT, Christmas
