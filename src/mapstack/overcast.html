Overcast Map Collection
A collection of publicly downloadable maps from the Overcast Network.

 Tip

You can include Cardinal tested maps from this repository on your Cardinal server by adding the following to the repos: section of the plugin configuration:

- git:https://github.com/MCResourcePile/overcast-cardinal-maps.git

Your repository will automatically update as more maps are added and tested with Cardinal ensuring that you only get functional maps. If you do find a map that isn't working as expected in the repository, please create an issue on our tracker.
